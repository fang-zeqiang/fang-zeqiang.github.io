在不确定性的河流中航行

一个数据工程师的政治经济世界观存

方泽强 / 2026-03-19

分类: 个人成长 / 标签: 政治经济, 技术革命, 反脆弱, 世界观, 不确定性 / 字数: 3971

「世界是非线性的。小原因可能引发大结果，极端事件主导历史，预测是徒劳的，生存优于优化。」——纳西姆·塔勒布

作为一个数据工程师，我的日常工作是处理确定性：清洗数据、建立模型、优化指标。但当我打开人生的广角镜头，看到的却是一片充满不确定性的汪洋。这篇文章是我近期对政治、经济、历史的一次系统性思考，试图在不确定性的河流中，找到航行的罗盘。

一、被隐藏的代价：工农剪刀差的历史回响

宏观视角：国家工业化的原始积累

1928年的苏联，斯大林面临着一道难题：如何在西方封锁的环境下，快速实现工业化？经济学家普列奥布拉任斯基给出了答案——「社会主义原始积累论」：通过行政手段压低农产品价格、抬高工业品价格，将农业剩余强制转移到工业部门。

这就是「工农剪刀差」——一把无形的剪刀，剪走了农民的利润，补贴了城市的工厂。

国家时期提取规模代价
苏联1928-1953占工业投资40%+乌克兰大饥荒(1932-33)，数百万人死亡
中国1953-20066000-8000亿元完整工业体系，但城乡二元结构固化
日韩台1960s-80s隐性城市偏向政策出口导向成功，农民阶层牺牲

微观案例：一个农民的账本

让我们看看这把剪刀是如何具体运作的。1953年，中国实行「统购统销」制度：国家以固定低价收购粮食，再以高价出售工业品。农民张大爷种了一亩地，产粮300斤：

  • 市场价格：粮食0.15元/斤，应得45元
  • 统购价格：粮食0.08元/斤，实得24元
  • 剪刀差：21元被提取，相当于47%的收入

这21元去了哪里？变成了沈阳机床厂的车床、鞍山钢铁厂的高炉、长春一汽的生产线。张大爷没有见过这些工厂，但他的汗水流淌在每一台机器里。

洞见：发展正义的永恒困境

诺贝尔经济学奖得主阿马蒂亚·森指出，剪刀差不仅是经济问题，更是「可行能力」的剥夺——农民失去了定价权、交易权、迁徙权。从国家视角看，这是快速工业化的必要代价；从分配正义视角看，这是对特定群体的系统性剥削。

这个困境至今仍在延续，只是形式变了。今天的「剪刀差」可能是房价、是教育成本、是996加班文化。发展从来不是免费的午餐，关键是：谁买单？谁受益？谁来补偿？

二、技术革命的潮汐：我们在历史周期的哪个位置？

宏观视角：佩雷斯的长波周期

技术经济学家卡洛塔·佩雷斯发现，工业革命以来，人类经历了五次技术革命，每次都遵循相同的模式：

技术突破 → 金融资本涌入 → 泡沫/狂热 → 崩溃/调整 → 制度重组 → 生产资本主导 → 黄金年代
次时期核心产业当前状态
11771棉纺织、铁、水力已成熟
21829蒸汽机、铁路已成熟
31875钢铁、电气已成熟
41908石油、汽车、大规模生产已成熟
51971ICT、互联网已成熟/衰退
6？2022？AI与智能革命导入期？

微观案例：运河热与铁路热中的普通人

历史不会重复，但会押韵。让我们回到1845年的英国，看看「铁路热」中的普通人经历了什么。

投资者：从理性到狂热

1845年，英国议会通过了2,700英里的新铁路计划。小学教师约翰在日记中写道：「我今天把积蓄全部买了『大北铁路』的股票。校长说我不理智，但他说的时候自己也买了。」

一年后，铁路股票暴跌80%。一个寡妇在破产申请中写道：「我丈夫留给我5,000英镑，我听从朋友建议投资铁路，现在只剩200英镑。」

建设者：隧道里的生命

25万铁路工人（占总劳动力2%）在挖掘隧道、铺设铁轨。一个工人在证词中说：「我们在挖一条穿过山丘的隧道。里面很黑，只有蜡烛。上周有3个人被塌方埋了，我们只挖出了2个。」

旁观者：被改变的生活

约克郡农民托马斯递交请愿书：「铁路要从我的农场中间穿过，把我的地切成两半。我的奶牛过不去了。」

当下的映射：AI热潮中的我们

历史角色今天对应风险
铁路股票投资者AI概念股投资者、加密货币投机者泡沫破裂，财富蒸发
铁路工人AI标注员、数据中心建设者、算法工程师工作伤害、职业替代
沿线居民被AI改变行业的从业者（翻译、设计、客服）生计被颠覆，技能过时

2023年，AI投资占全球VC的30%以上，OpenAI估值800亿美元。我们正处于佩雷斯框架中的「狂热阶段」（Frenzy）——金融资本主导，技术不确定性高，商业模式不清晰。

接下来的剧本可能是：

  • 2025-2027：泡沫破裂，部分高估值AI公司倒闭
  • 2027-2028：重大AI安全事故，引发监管危机
  • 2028-2040+：制度重组，生产资本主导，可能的「黄金年代」

洞见：这一次是否不同？

AI革命可能有四个不同：

  1. 速度压缩：周期从50-60年压缩至20-30年
  2. 奇点风险：如果AGI在导入期就实现，历史模式可能失效
  3. 地缘政治：中美AI脱钩可能导致「两个技术-经济范式」
  4. 分配挑战：AI可能永久性替代认知劳动，如果制度调整失败，可能长期停滞

三、置身事内：中国政府的「发展型国家」角色

宏观视角：地方政府公司化

经济学家兰小欢在《置身事内》中提出一个核心洞见：理解中国经济，不能用「政府vs市场」的二分法。中国政府不仅是裁判员，更是运动员——深度参与经济，直接推动发展。

关键制度演变（四部曲）：

时期核心机制结果
1994前财政包干中央弱、地方强
1994分税制改革中央集权，土地财政兴起
1998-2020土地财政+城投债城市化快、债务累积
2020后双循环+房住不炒旧模式难持续，转型中

土地财政的逻辑：

低价征地（农村）→ 高价出让（城市）→ 差价用于基建 → 地价上涨 → 更多抵押

这本质上是剪刀差的现代形式——农村剩余的城市化提取。1998-2020年间，地方政府通过这种方式完成了人类历史上最大规模的城市化。

微观案例：合肥投资京东方

2008年，合肥市政府决定投资京东方建设第6代液晶面板生产线。当时合肥的财政收入只有300亿元，而项目投资需要175亿元。

市长亲自带队，通过城投公司融资、土地抵押、银行贷款，最终项目落地。今天，京东方是全球最大的液晶面板生产商，合肥也因此成为「显示之都」。

这是一个成功的「发展型政府」案例，但也有失败的案例——光伏补贴导致产能过剩，但客观上推动了技术迭代。

洞见：情境依赖的产业政策

产业政策的有效性高度依赖具体执行情境：

  • 成功条件：地方政府有动力、有资源、有执行力
  • 失败风险：信息不对称、道德风险、债务累积
  • 关键变量：中央与地方的激励相容、官员考核机制

四、反脆弱生存：在不确定性的河流中航行

宏观视角：从预测到准备

塔勒布在《黑天鹅》《反脆弱》中提出了一个核心洞见：复杂系统的未来不可预测，只能准备。

传统思维：「我认为明年股市会涨10%，所以满仓股票。」

塔勒布思维：「我不知道明年会发生什么，但我知道：

  • 如果崩盘，我有现金可以抄底
  • 如果通胀，我有实物资产保值
  • 如果繁荣，我的冒险仓位会爆发
  • 如果停滞，我的安全仓位让我活下去」

微观案例：杠铃策略的实践

塔勒布建议的「杠铃策略」：90%极度安全 + 10%高风险高回报，避免中间地带。

我的个人配置：

仓位配置目的
90%安全指数基金、国债、现金确保生存底线
10%冒险期权、早期项目、技能学习有限损失，无限 upside

避免：60%股票+40%债券的「中等风险」组合——看似平衡，实则同时暴露于市场崩盘和通胀侵蚀。

建立「选择权」（Optionality）

类型例子意义
财务期权持有现金等待机会有选择不工作的权利
时间期权延迟决策，等待更多信息有选择不立即行动的权利
知识期权学习多种可迁移技能有选择转行/创业的权利
职业期权副业收入 + 低成本生活有选择不接受烂工作的权利

洞见：演化思维的人生决策

从「规划」到「演化」：

决策领域传统思维演化思维
职业线性晋升（专员→经理→总监）技能组合多样化，保持可迁移性
技能专精一门T型或π型：广度+深度
城市定居一处核心城市「网络节点」效应
关系固定圈子弱连接维护（创新来源）
健康目标体重/指标系统韧性（睡眠、运动、压力的动态平衡）

核心洞察：「不是成为最好的，而是成为「最可进化的」——在变化的环境中，适应能力比当前适应度更重要。」

五、编织认知网络：我的世界观罗盘

通过这段时间的思考，我试图编织一个认知网络，帮助自己在不确定性的河流中航行：

┌─────────────────────────────────────────────────────────┐
│                    演化思维的三个维度                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   《规模》（West）        《技术革命与金融资本》（Perez）    │
│      ↓                        ↓                        │
│   生物/城市/公司的        技术-经济范式的               │
│   亚线性/超线性缩放       周期性演化                   │
│      ↘                    ↙                           │
│        → 复杂系统的自组织与涌现 ←                      │
│                      ↓                                 │
│              演化思维的核心洞见                         │
│         「没有最优，只有适应；没有均衡，只有演化」        │
│                                                         │
└─────────────────────────────────────────────────────────┘

三个核心原则：

  1. 历史周期意识：我们正处于第六次技术革命的导入期，狂热与泡沫是常态，关键是生存到协同期
  2. 反脆弱生存：不预测，只准备；建立选择权；杠铃策略配置资源
  3. 演化适应：保持可进化性，T型技能发展，维护弱连接网络

结语：在不确定性的河流中

「你无法管理你不衡量的东西，但衡量本身不是目的。目的是通过数据，看见真实的自己，然后温柔地调整。」

作为一个数据工程师，我习惯了确定性。但政治、经济、历史告诉我，世界本质上是不确定的。我们能做的，不是预测河流的流向，而是建造一艘能在各种水流中生存的船。

这艘船的名字，叫做「反脆弱」。
