在不确定性的河流中航行：一个数据工程师的生存指南

方泽强 / 2026-03-19

作为一个每天和确定性打交道的人，我发现世界本质上是不确定的。

我的工作是处理数据：清洗、建模、优化指标。数字不会撒谎，公式不会背叛你。但当我关掉电脑，打开人生的广角镜头，看到的却是一片充满不确定性的汪洋。

这篇文章，是我作为一个数据工程师，对当下时代的思考，也是给自己的一份生存指南。

一、我们正站在什么位置？

技术经济学家卡洛塔·佩雷斯发现，工业革命以来，人类经历了五次技术革命，每次都遵循相同的剧本：

技术突破 → 资本涌入 → 泡沫狂热 → 崩溃调整 → 制度重组 → 黄金年代

前五次分别是：纺织业、铁路、电气、汽车、互联网。它们都已经走完了完整的周期。

而第六次，可能正在发生——AI与智能革命。

2023年，AI投资占全球VC的30%以上，OpenAI估值800亿美元。我们正处于「狂热阶段」：金融资本主导，技术不确定性极高，商业模式尚不清晰。

这让我想起1845年的英国铁路热。

那时，英国议会一年通过了2700英里的铁路计划。小学教师把积蓄全部投入铁路股票，一年后暴跌80%。25万工人在隧道里挖掘，有人被塌方掩埋。农民的土地被切成两半，奶牛再也过不去。

历史不会重复，但会押韵。

今天的「铁路股票投资者」，是AI概念股和加密货币的投机者；今天的「铁路工人」，是AI标注员和算法工程师；今天的「沿线居民」，是被AI改变行业的从业者——翻译、设计、客服，以及未来可能被波及的所有知识工作者。

问题是：在这个剧本里，你扮演谁？

二、这一次，可能真的不同

AI革命与以往有四点关键不同：

第一，速度压缩。 过去的周期需要50-60年，这次可能压缩到20-30年。留给我们适应的时间更少了。

第二，奇点风险。 如果通用人工智能（AGI）在导入期就实现，历史模式可能完全失效。没人知道会发生什么。

第三，地缘政治。 中美AI脱钩可能导致「两个技术-经济范式」，全球化的协同效应被削弱。

第四，也是我最担心的——分配挑战。

AI可能永久性替代认知劳动。如果制度调整跟不上，我们可能面临长期的经济停滞和社会撕裂。这不是危言耸听，而是数据告诉我的可能性。

作为一个数据工程师，我习惯用模型预测未来。但这次，我越来越觉得：预测是徒劳的，准备才是正解。

三、第一层：反脆弱——建造不沉的船

纳西姆·塔勒布在《反脆弱》中提出了一个核心洞见：复杂系统的未来不可预测，只能准备。

传统思维是：“我认为明年股市会涨10%，所以满仓股票。”

反脆弱思维是：“我不知道明年会发生什么，但我知道——

  • 如果崩盘，我有现金可以抄底
  • 如果通胀，我有实物资产保值
  • 如果繁荣，我的冒险仓位会爆发
  • 如果停滞，我的安全仓位让我活下去”

这叫杠铃策略：90%极度安全 + 10%高风险高回报，彻底避开中间地带。

我的个人配置是这样的：

仓位配置目的
90%安全指数基金、国债、现金确保生存底线
10%冒险期权、早期项目、技能学习有限损失，无限可能

我刻意避免那种"60%股票+40%债券"的所谓平衡组合——看似稳健，实则同时暴露于市场崩盘和通胀侵蚀。

但反脆弱不只是金钱配置，更是选择权的积累。

塔勒布说，真正的财富不是数字，而是「选择权」（Optionality）。我给自己建立了四种选择权：

选择权我的实践意义
财务期权持有足够现金有选择不工作的权利
时间期权延迟重大决策有选择等待信息的权利
知识期权学习可迁移技能有选择转行/创业的权利
职业期权副业+低成本生活有选择不接受烂工作的权利

反脆弱解决的是底线问题：无论河流如何湍急，我的船不会沉。

但这够了吗？

四、第二层：演化思维——学会在任何水流中航行

反脆弱让我活下来。但如果只是活下来，却永远在原地打转，那也不是我想要的。

演化思维解决的是上限问题：不仅要活着，还要活得好。

过去的我，喜欢做五年规划：专员→经理→总监，线性晋升，步步为营。我以为只要规划得足够精密，就能抵达确定的未来。

现在我知道，这是幻觉。

从规划到演化，我改变了五个维度：

维度规划思维（过去）演化思维（现在）
职业路径线性晋升（专员→经理→总监）技能组合多样化，保持可迁移性
能力结构专精一门技术T型或π型：广度+深度
地理选择定居一处核心城市「网络节点」效应
社交网络固定圈子维护弱连接（创新的来源）
健康目标目标体重/指标系统韧性（睡眠、运动、压力的动态平衡）

核心洞察：不是成为最好的，而是成为「最可进化的」。

在变化的环境中，适应能力比当前适应度更重要。恐龙很强壮，但灭绝了；蟑螂很普通，但活到了今天。

五、两层的关系：底线与上限

反脆弱和演化思维，是递进的两层：

        ┌─────────────────────┐
        │    演化思维         │
        │  （如何活得好）      │
        │  可进化性 > 当前适应度 │
        └──────────┬──────────┘
                   │ 递进：先保住底线
                   │ 再追求上限
                   ▼
        ┌─────────────────────┐
        │    反脆弱           │
        │  （如何活下来）      │
        │  杠铃策略 + 选择权   │
        └─────────────────────┘

第一层（反脆弱）是必要条件：没有它，任何演化都是空谈。船沉了，再好的航海术也没用。

第二层（演化思维）是充分条件：有了它，反脆弱才有意义。船不沉，但永远在原地打转，也不是我想要的。

两者的共同点是：都放弃了对未来的确定性假设。

  • 反脆弱说：我不预测，我只准备
  • 演化思维说：我不规划，我只适应

六、我的世界观罗盘

这段时间的思考，让我编织了一个简单的认知网络：

              历史周期意识
                    ↓
        「我们正处于AI革命的导入期」
                    ↓
        狂热与泡沫是常态，关键是生存到协同期
                    ↓
        ┌─────────────────────────┐
        │      第一层：反脆弱      │
        │   建造不沉的船（底线）   │
        │   杠铃策略 + 四种选择权  │
        └────────────┬────────────┘
                     │ 递进
                     ▼
        ┌─────────────────────────┐
        │      第二层：演化思维    │
        │   学会在任何水流中航行   │
        │   可进化性 > 当前适应度  │
        └─────────────────────────┘

这三个原则，是我在不确定性河流中的罗盘。

结语

“你无法管理你不衡量的东西，但衡量本身不是目的。目的是通过数据，看见真实的自己，然后温柔地调整。”

作为一个数据工程师，我习惯了确定性。但政治、经济、历史告诉我，世界本质上是不确定的。

我们能做的，不是预测河流的流向，而是：

  1. 建造一艘不沉的船（反脆弱）
  2. 学会在任何水流中航行（演化思维）

这艘船的名字，叫做「反脆弱」。而驾驶它的能力，叫做「演化」。

写于 2026 年 3 月 19 日，上海。 如果你也在建造自己的船，欢迎交流。
